    triangle

    @mixin triangle($direction, $width, $height, $color) { ... }

    Description

    Generates a triangle pointing in a specified direction.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $direction

    The direction the triangle should point. Accepts up, up-right, right, down-right, down, down-left, left or up-left.

    		String— none
    $width

    Width of the triangle.

    		Number (with unit)— none
    $height

    Height of the triangle.

    		Number (with unit)— none
    $color

    Color of the triangle.

    Usage:

    		Color— none

    Example

    .element {
  &::before {
    @include triangle("up", 2rem, 1rem, #b25c9c);
    content: "";
  }
}

// CSS Output
.element::before {
  border-style: solid;
  height: 0;
  width: 0;
  border-color: transparent transparent #b25c9c;
  border-width: 0 1rem 1rem;
  content: "";
}

    Throws

    • Direction must be up, up-right, right, down-right,

    • #{$color} is not a valid color for the $color argument in

  default: $typography-default-stack

    Description

    Mapowanie typografii względem RWD Dzięki tej tablicy możemy dla poszczególnych wielkości wskazać konkretną typografie Jako klucz nazwa urządzenia, która się pokrywa z $breakpoints, lub wartość w px (default jest wartością domyślną) Jako wartość podajemy zbiór z określoną typografią.
